
# 📦 Inventory Management System

A modern, feature-rich Inventory Management System built with pure HTML, CSS, and JavaScript. Features glassmorphism UI, real-time charts, IST time support, and comprehensive reporting.

![Version](https://img.shields.io/badge/version-2.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)
![Status](https://img.shields.io/badge/status-active-success.svg)

## 🌐 Live Demo

🔗 **[Visit Live Site](https://inventory.aitoolcor.com)**

## ✨ Features

### 🔐 Authentication
- Secure login system
- Multi-user support (Admin, Manager, Staff)
- Remember me functionality
- Session management
- Password change

### 📊 Dashboard
- Real-time statistics
- Animated counters
- Growth indicators (↑↓)
- Multiple charts (Line, Doughnut, Bar)
- Category distribution
- Top products by value
- Purchase trends
- Date range filters (Today/Week/Month/Year/Custom)

### 📦 Inventory Management
- Add/Edit/Delete products
- Category management
- Stock tracking with alerts
- Search & filter
- Multiple units support

### 🛒 Purchase Management
- Create purchase orders
- Multi-product invoices
- Auto stock update
- Payment tracking
- Due date management

### 📤 Stock Out Management
- Track outgoing stock
- Multiple reasons
- Auto stock deduction

### 🚚 Supplier Management
- Supplier database
- Purchase & payment history
- Pending invoices tracking

### 📈 Reports
- 5 types of reports
- Excel export
- PDF export

### ⏰ IST Time Support
- All timestamps in Indian Standard Time
- Live clock display
- Daily activity logs

## 🎨 Design

- 🪟 Glassmorphism UI
- 🌈 Soft pastel gradients
- 💫 Smooth animations
- 📱 Fully responsive

## 🛠 Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Chart.js for charts
- jsPDF & SheetJS for exports
- Font Awesome for icons
- Google Fonts (Poppins)
